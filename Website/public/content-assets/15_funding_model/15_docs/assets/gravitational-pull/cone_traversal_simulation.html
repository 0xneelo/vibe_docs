Parameter Guide

Gravity (0-2)
Controls what % of balls can move upward AND pulls them inward toward center.
0 = 0% up 1 = 50% up 2 = 100% up
Z-score Limit (0-2)
Maximum height that activated balls can reach.
0 = 0% 1 = 50% 2 = 100%
H-spread (0-2)
Horizontal distribution. Spreads balls evenly across the X/Y plane.
0 = clustered 2 = even spread
V-spread (0-2)
Vertical distribution. Spreads balls evenly along the Z axis.
0 = all at top 2 = even spread
Outlier % (0-60)
Percentage of balls with special behaviors (rebels, wanderers, stragglers, etc).
