# Section 4: The Assembly Line

The most important contribution is not a single market design. It is the assembly line.

## 4.1 Protocol-Defined Lifecycle

The proposed path is simple:

```text
LAUNCHPAD -> DEX -> VIBE PERP -> ORDER BOOK
```

A token launches on a bonding curve, graduates to a DEX, and is immediately listed on Vibe as a perpetual. At that point, the market gains both spot and perp trading. If sufficient liquidity, volume, and trader diversity emerge, the market can then graduate into an order book venue such as Hyperliquid.

## 4.2 Why This Matters

Without this assembly line, every stage is disconnected. Creation is permissionless, but derivatives are permissioned. Spot is abundant, but shorting is scarce. Demand is noisy, but listing decisions are expensive.

With the assembly line, every stage becomes legible:

- launchpads create assets
- DEXs create spot access
- Vibe creates the first true derivatives market
- order books absorb the winners

That is what a mature financial system looks like: not one venue doing everything, but multiple layers connected by clear rules.
