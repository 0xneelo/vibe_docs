# 10 - Vibe Pillars

Draft workspace for a paper on the core structural pillars of Vibe.

This folder develops a simple thesis: permissionless perpetuals for the long tail only work if three hard problems are solved together:

- exploit resistance
- market bootstrap / guaranteed counterparty access
- sufficient LP yield through capital efficiency

The main source material is the rough transcript in `13_transcripts/13_transcript.md`, expanded into a cleaner paper structure in `13_docs/`.
