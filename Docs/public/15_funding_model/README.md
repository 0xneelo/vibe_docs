# 15 - Funding Rate Model

A technical chapter for the funding-rate engine, including formal derivation, defensive controls, and scenario analysis.

This chapter explains how funding, utilization stress, and emergency dynamics interact to protect solver capital while preserving market continuity. It also outlines the assumptions behind recovery behavior and risk controls across volatile conditions.
