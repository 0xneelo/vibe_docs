# 11 - Game Theory of Listings

Draft workspace for a paper on how listing mechanisms shape crypto market structure.

The core thesis is that any listing venue with non-trivial cost, curation, or reputational exposure must optimize around **perceived interest**, not **actual interest**. That creates structural distortions across the industry: low-float high-FDV launches, metric gaming, over-curated listings, and a widening disconnect between where tokens are created and where they can access serious derivatives.

This folder develops the case that zero-cost permissionless listing systems preserve evolutionary discovery better than curated or high-cost systems, and that Vibe's role is to bring perpetuals to the `zero perceived interest` arena where future unicorns are born.
