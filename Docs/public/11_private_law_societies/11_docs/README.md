# Private Law Societies

## Overview

This folder contains research and draft materials for the **Sovereign Intent Protocol** and the broader **private law societies** thesis.

Rather than acting as a polished paper itself, this `README.md` is now the clean entry point into the material.

## Structure

| Path | Purpose |
|------|---------|
| `11_whitepaper/` | Chapter-by-chapter whitepaper draft |
| `11_sources/` | Research sources, summaries, translations, and PDFs |
| `11_website/` | Canonical web app / presentation build |
| `SovereignIntentProtocol_Whitepaper.md` | Long-form single-file whitepaper draft |

## Suggested Reading Order

1. `11_whitepaper/00-front-matter.md`
2. `11_whitepaper/01-executive-summary.md`
3. `11_whitepaper/02-introduction.md`
4. `11_whitepaper/03-failure-monopolistic-counterparty.md`

## Notes

- The `11_sources/` folder contains both summaries and source/reference documents.
- The single-file `SovereignIntentProtocol_Whitepaper.md` appears to be the consolidated master draft.
