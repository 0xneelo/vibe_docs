# APPENDIX C: GLOSSARY OF TERMS

| Term | Definition |
|------|------------|
| **Bilateral Isolation** | Property where failure of one contract does not affect others |
| **CVA** | Credit Valuation Adjustment; locked collateral guaranteeing performance |
| **Futarchy** | Governance by prediction markets rather than voting |
| **Intent** | Citizen's declaration of desired outcome (e.g., protection, dispute resolution) |
| **Intent Aggregator** | Entity that bundles common protection needs into template policies, negotiating bulk rates with PSAs and Courts on behalf of subscribers |
| **NAP** | Non-Aggression Principle; prohibition on initiated force |
| **Outlaw** | Individual who has exited the contractual order via non-compliance or by choice |
| **Panarchy** | Coexistence of multiple governance systems with free choice |
| **PSA** | Private Security Agency; competing private provider of protection/enforcement services |
| **Private Court** | Competing private institution providing dispute resolution with employed or affiliated judges |
| **Arbiter** | Independent private judge who stakes personal capital; slashed for corrupt rulings |
| **PRG** | Privatrechtsgesellschaft; Private Law Society |
| **Restitution** | Compensation to victim from violator's resources |
| **SSIP** | Symmio Sovereign Intent Protocol |
| **Solver** | Provider who fulfills citizen intents (SYMMIO terminology) |
| **SYMMIO** | Bilateral OTC Derivatives Protocol; technical infrastructure for Symmetrical contracts |
| **Warlord** | Entity attempting to establish coercive territorial monopoly |
| **ZEDE** | Zone for Employment and Economic Development; Honduran autonomous zones |
