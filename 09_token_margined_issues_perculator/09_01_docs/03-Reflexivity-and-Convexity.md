# Section 3: Reflexivity and Negative Convexity

## 3.1 The Reflexivity Problem

### 3.1.1 Definition

Reflexivity occurs when a price movement simultaneously affects both the position's PnL and the collateral's value. In token-margined systems, collateral and exposure share correlation **1.0**.

### 3.1.2 The Double Hit

Consider a trader holding a long position on Token X, collateralized in Token X:

1. Token X drops 10%
2. Position loses value (PnL impact)
3. Collateral also drops 10% (margin impact)

The effective loss to margin ratio is **compounded**, not additive. In USDC-margined systems, only the position moves; collateral stays stable.

**Implications**:
- Liquidation happens faster
- Margin buffers shrink precisely when needed
- Higher maintenance margins required for same safety
- Capital efficiency reduced

### 3.1.3 Systemic Reflexivity

When the entire market uses the same asset as collateral:

```
Price drops → Collateral devaluation → Margin ratio collapse →
Forced liquidations → Selling pressure → Further price drops → ...
```

This feedback loop destroyed multiple DeFi protocols during the LUNA crash (May 2022) and March 2020 cascades.

### 3.1.4 Endogenous Risk

Permissionless listing + same-asset collateral creates **endogenous risk**: risk generated by the system itself. The protocol's liquidation mechanics become a volatility source, amplifying moves that would otherwise be benign.

## 3.2 Negative Convexity and the Inverse Payoff Trap

### 3.2.1 The Hyperbolic Payoff

Inverse perpetuals use:

$$
\text{PnL}_{\text{tokens}} = \text{Contracts} \times \left(\frac{1}{P_{\text{entry}}} - \frac{1}{P_{\text{exit}}}\right)
$$

**Asymmetric profile in token terms**:
- **Upward moves**: Long PnL bounded (approaches limit as price → ∞)
- **Downward moves**: Short PnL **unbounded** (approaches ∞ as price → 0)

### 3.2.2 The Infinity Payout Problem

Short position on $PERC entered at $1.00:

| Exit Price | USD Profit | Token Payout Required |
|------------|------------|------------------------|
| $0.50 | $0.50 | 1.0 PERC |
| $0.10 | $0.90 | 9.0 PERC |
| $0.01 | $0.99 | 99.0 PERC |
| $0.001 | $0.999 | 999.0 PERC |

As price approaches zero, the protocol must pay **exponentially more tokens** for the same USD profit. A single large short during a 99% crash can drain the vault—token-denominated liability grows hyperbolically as the token loses value.

### 3.2.3 Contrast with Linear Systems

USDC-margined: same 99% crash creates **fixed USD liability**. Trader shorts $1,000 at $1.00; price drops to $0.01; profit = $990 USDC (fixed, bounded, payable). Payout does not expand. Vault's ability to pay is independent of asset price.

---

## 3.3 Why Linear Settlement Wins

| Scenario | Percolator (Inverse) | USDC-Margined (Linear) |
|----------|----------------------|-------------------------|
| Token drops 90% | Short payout: 9× contract size in tokens; vault hemorrhages | Short payout: 90% of notional in USDC; fixed, payable |
| Token pumps 10× | Long payout bounded; LP still loses tokens | Long payout: 9× notional in USDC; large but linear |
| Flash crash to near-zero | Infinite token liability; vault drained; ADL forced | Fixed USDC liability; bounded; insurance can cover |

The linear payoff eliminates the entire class of negative convexity risks that make token-margined systems structurally fragile.
